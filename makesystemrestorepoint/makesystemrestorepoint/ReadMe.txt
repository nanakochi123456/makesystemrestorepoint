========================================================================
    Win32 アプリケーション: makesystemrestorepoint プロジェクトの概要
========================================================================

この makesystemrestorepoint アプリケーションは、AppWizard により作成されたなの。

このファイルには、makesystemrestorepoint アプリケーションを構成する各ファイルの内容の概要が含まれているなの。


makesystemrestorepoint.vcxproj
    これは、アプリケーション ウィザードを使用して生成された VC++ プロジェクトのメイン プロジェクト ファイルなの。ファイルを生成した Visual C++ のバージョンに関する情報と、アプリケーション ウィザードで選択されたプラットフォーム、構成、およびプロジェクト機能に関する情報が含まれているなの。

makesystemrestorepoint.vcxproj.filters
    これは、アプリケーション ウィザードで生成された VC++ プロジェクトのフィルター ファイルなの。このファイルには、プロジェクト内のファイルとフィルターとの間の関連付けに関する情報が含まれているなの。この関連付けは、特定のノードで同様の拡張子を持つファイルのグループ化を示すために IDE で使用されます (たとえば、".cpp" ファイルは "ソース ファイル" フィルターに関連付けられているなの)。

makesystemrestorepoint.cpp
    これは、メインのアプリケーション ソース ファイルなの。

/////////////////////////////////////////////////////////////////////////////
AppWizard によって、次のリソースが作成されたなの。

makesystemrestorepoint.rc
    これは、プログラムが使用するすべての Microsoft Windows リソースの一覧なの。RES サブディレクトリに格納されるアイコン、ビットマップ、およびカーソルをインクルードするなの。このファイルは、Microsoft Visual C++ で直接編集できるなの。

Resource.h
    これは、新しいリソース ID を定義する標準のヘッダー ファイルなの。このファイルの読み込みおよび更新は、Microsoft Visual C++ で行うなの。

makesystemrestorepoint.ico
    これは、アプリケーションのアイコン (32x32) として使用されるアイコン ファイルなの。このアイコンは、メイン リソース ファイル makesystemrestorepoint.rc にインクルードされるなの。

small.ico
    これは、アプリケーションのアイコンの小さいバージョン (16x16) を含むアイコン ファイルなの。このアイコンは、メイン リソース ファイル makesystemrestorepoint.rc にインクルードされるなの。

/////////////////////////////////////////////////////////////////////////////
その他の標準ファイル :

StdAfx.h, StdAfx.cpp
    これらのファイルは、makesystemrestorepoint.pch という名前のプリコンパイル済みヘッダー (PCH) ファイルと、StdAfx.obj という名前のプリコンパイル済みの型ファイルをビルドするために使用されるなの。
